                        # 简介

                        # 背景

                        • 减少重复代码逻辑，将工作集中于业务逻辑上
                        • 开发快捷高效，将时间用于业务代码上
                        • 统一审批，便于后期维护

                        # 使用

                        • js中引入组件 import ApprovalProcess from "/@/components/ApprovalProcess";
                        • html中注入组件 <ApprovalProcess />

                        # 属性

                        # ApprovalProcess属性

                        属性名默认值类型描述用法
                        title"审核进度"String标题title="审核进度"
                        isShowExamineBtnfalseBoolean是否展示页面底部通过、驳回按钮:isShowExamineBtn="true"
                        queryAuditNode() => nullFunction是一个function,调用接口，需返回一个对象，包含审批节点数据nodeItems、审批表格taskComments、taskId参考下方源代码
                        approvalEditfalseBoolean审批弹窗中的审核结果字段，是否展示可编辑:approvalEdit="true"

                        # ApprovalProcess事件

                        事件名说明回调参数
                        callBackAudit审批回调，拿到params参数，发送审批回调请求，并修改isShowExamineBtn值，隐藏底部操作按钮params对象， 包含4个属性：auditStatus：审核结果，2同意，3驳回；comment：审核意见；isPc: 1；taskId: ''
